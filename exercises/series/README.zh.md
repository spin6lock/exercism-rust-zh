# 系列

给定一串数字,输出所有连续的`n`长度顺序子串。

例如,字符串"49142"长度为`3`的子串系列:

- "491"
- "914"
- "142"

以下长度为 `4` 的 系列:

- "4914"
- "9142"

如果你要求一个 5 位数字，长度为 6 的系列,你应该得到空。

请注意,这些系列只需要在输入中是*相邻的位置*; 数字不需要在*数字上连续*.
